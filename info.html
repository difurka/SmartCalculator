Info about CALCULATOR

Programm is created for calculating expressions with x, drawing graphics and making credits.

  • You can calculate expressions with x when define x.

    If you make any mistake the answer will be "NAN".

  • You can drawing graphics. The plot is appiares in a new window.

    If you make any mistake the plot will be empty.

  • You can calculate credits.

    If you make any mistake the answer will be ERROR.

Enjoy with CALCULATOR!!!
